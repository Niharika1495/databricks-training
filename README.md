# Databricks Training

This repository contains my hands-on practice, assignments, and learning outcomes from the Databricks Training Program conducted by our college. The repository is organized systematically to track daily progress and improve SQL, Databricks, and data engineering concepts step by step.

## Repository Structure

```text
Week1/
│
└── Day1/
    ├── schema.sql
    ├── queries.sql
    ├── README.md
    └── outputs/
```

## Files Description

* **schema.sql**
  Contains database schema creation scripts, table definitions, constraints, and sample data insertion queries.

* **queries.sql**
  Includes SQL practice queries covering:

  * Basic Queries
  * String Matching
  * Date Functions
  * Aggregate Functions
  * GROUP BY & HAVING
  * ORDER BY
  * JOIN Operations
  * Nested & Correlated Queries

* **outputs/**
  Stores query execution screenshots or output files for reference and documentation.

* **README.md**
  Provides an overview of the work completed and repository structure.
