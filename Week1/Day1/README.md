# Week 1 - Day 1

## Objective

The objective of this session was to understand the fundamentals of SQL and gain hands-on experience in creating tables, inserting data, and retrieving information using different SQL queries.

---

## Tasks Completed

- Created relational database tables using SQL schema definitions
- Inserted sample records into the tables
- Executed multiple SQL queries for practice
- Verified query outputs for correctness
- Organized files in a structured repository format

---

## Topics Covered

### Basic SQL Operations
- SELECT statements
- Filtering records using `WHERE`
- Sorting data using `ORDER BY`

### String & Date Operations
- Pattern matching using `LIKE`
- Working with date conditions and filters

### Aggregate Functions
- `COUNT()`
- `SUM()`
- `AVG()`
- `MIN()`
- `MAX()`

### Advanced Concepts Introduced
- GROUP BY
- HAVING clause
- Basic JOIN operations
- Introduction to nested queries

## Learning Outcomes

- Understood how relational databases are structured
- Learned to write and execute SQL queries
- Gained experience with filtering, sorting, and aggregation
- Improved understanding of joins and grouped queries
- Learned how multiple SQL clauses work together in real scenarios

Initially, understanding conditions inside `WHERE` clauses and combining multiple SQL statements was slightly challenging. After practicing several queries and analyzing outputs, the concepts became much clearer.

Successfully completed Week 1 - Day 1 tasks and built a strong foundation in SQL basics. This session improved confidence in writing queries and prepared the groundwork for learning more advanced SQL and Databricks concepts in upcoming sessions.
